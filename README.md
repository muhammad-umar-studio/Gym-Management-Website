# Gym-Management-Website

# Proccess to access the file
1. Run the App.py file. <br>
2. Now open the link which shown in the terminal.<br>
3. Open this link in the browser (http://127.0.0.1:5000).<br>
4. If you open this link (http://127.0.0.1:5000/about) the about page will be open.<br>
5. When you open this link(http://127.0.0.1:5000/membership) the membership page will be open.<br>
